# Dustin Muse - Portfolio Website

Welcome to the source code for my personal portfolio website! This project serves as a showcase of my skills, projects, and professional experience as a Software Developer.

## 🚀 Overview

This website is a fully responsive and modern static web application built to highlight my background in Computer Science and my passion for software engineering. It features a premium dark theme, glassmorphism design elements, vibrant accents, and smooth scroll animations.

## 🛠️ Built With

- **HTML5:** For semantic structure and accessibility.
- **CSS3:** For styling, including CSS variables, flexbox, CSS grid, and modern techniques like glassmorphism and animated abstract background shapes.
- **JavaScript (Vanilla):** For interactive elements, specifically utilizing Intersection Observers for clean scroll reveal animations.
- **FontAwesome:** For intuitive iconography.
- **Google Fonts:** Utilizing 'Inter' and 'Outfit' for clean, modern typography.

## 🌟 Features

- **Responsive Design:** Ensures a seamless experience across all devices (desktop, tablet, and mobile).
- **Hero Section:** A visually appealing introduction with dynamic gradient text, floating background elements, and calls to action.
- **Education & Experience Timeline:** Highlights my academic journey at Ball State University and my professional background (including Teaching Assistant roles).
- **Skills Showcase:** Visual representation of my technical proficiencies in Front-End, Back-End, and Database technologies using animated progress bars.
- **Featured Projects:** A curated list of my key projects (like AI Resume Scanner, Airline Booking System, and various Java networking/OS projects) with direct links to their repositories.
- **Contact Page:** A dedicated, easy-to-navigate page to connect with me via Email, LinkedIn, or GitHub.

## 📂 Project Structure

- `index.html`: The main landing page containing the Home, Education, Skills, Projects, and Experience sections.
- `contact_me.html`: A dedicated page containing my contact information and professional links.
- `style.css`: The central stylesheet containing all design tokens, responsive media queries, and keyframe animations.
- `images/`: Directory containing visual assets used across the site.

## 💻 Local Development

To run this project locally, simply clone the repository and open the main HTML file in your web browser. No complex build tools or dependencies are required.

```bash
# Clone the repository
git clone https://github.com/Dustinmuse/Portfolio-website.git

# Navigate to the project directory
cd Portfolio-website

# Open index.html in your default web browser
```
